Sphinx Domain for documenting SQL databases

Overview
--------

This Domain will contain Directives for:

- Database
- Schema
- Table
- View
- Column
- Type
- Procedure
- Index
- Constraint
- Primary Key
- Grant
- User
- Group/Layer (undecided)
