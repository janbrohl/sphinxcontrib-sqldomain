###########################
  sphinxcontrib-sqldomain
###########################

This package contains sphinxcontrb.sqldomain, a Sphinx Domain for documenting SQL databases.

This Domain will contain Directives for:

- Database
- Schema
- Table
- View
- Column
- Type
- Procedure
- Index
- Constraint
- Primary Key
- Grant
- User
- Group/Layer (undecided)
